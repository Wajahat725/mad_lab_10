# mad_lab_10

=============================================Product Catalog==============================================
                                                                            
I have built product Catalog application in flutter that demonstrates the implementation of essential Flutter widgets in an e-commerce context.

 Features
 --------
 Browse Products - Explore products from various categories.

 Category Filtering - Filter products by categories (Electronics, Fashion, Home, etc.).

 Wishlist Management - Add and manage favorite products.

 Product Search - Find products quickly with search functionality.

 Professional UI - Modern, clean e-commerce design with smooth animations.

 Responsive Layout - Optimized for different screen sizes.

 Efficient Scrolling - Smooth performance with large product catalogs.
